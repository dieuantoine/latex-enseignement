# Cours de Physique-Chimie Collège

Ensemble des documents de cours sur la période février-juin 2026.

Auteur : M. Dieu, Collège Charles Péguy

## Contenu

templates : fichiers .sty qui définissent la mise en page des documents et importent les packages utiles.

Les dossiers 6e, 5e et 3e contiennent un sous-dossier par document. Les noms de fichiers suivent le format suivant :
- Le premier caractère indique la classe
- Le deuxième caractère indique le chapitre :
    - C -> Chimie,
    - E -> Electricité,
    - M -> Mouvement,
    - S -> Signal,
    - W -> Energie,
    - A -> Autre
- Le troisième caractère indique le numéro du document dans le chapitre
- Le quatrième caractère indique le type de document :
    - C -> Cours complet,
    - D -> Document de cours
    - E -> Evaluation,
    - F -> Fiches
    - I -> Séance d'introduction
    - X -> Feuille d'exercices
